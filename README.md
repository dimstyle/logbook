# Employee Attendance Logbook

A web-based attendance and daily activity logbook system designed to help employees record their attendance and daily work activities.

> 🚧 This project is currently under development.

## 📌 Planned Features

- User Authentication
- Daily Check In
- Daily Activity Report
- Daily Check Out
- Attendance History
- User Profile
